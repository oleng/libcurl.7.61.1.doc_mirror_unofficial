Here is a list of libraries to help you chew up what you catch with libcurl.

Name Description
liburi library to analyze and transform URI's
ftpparse library for parsing FTP LIST responses.
libtidy html parsing and formatting library, based on Dave Raggett's HTML-Tidy
libmagic library interface to the unix "file" command, to determine the filetype of a file or buffer
libxml extensive XML parsing library
